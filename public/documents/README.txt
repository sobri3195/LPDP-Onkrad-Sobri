INSTRUKSI PENGISIAN DOKUMEN PDF
================================

Folder ini adalah tempat untuk menyimpan semua dokumen PDF yang akan diunduh dari website.

DOKUMEN YANG DIPERLUKAN:
------------------------

1. LPDP Documents:
   - Motivation_Letter.pdf
   - Career_Plan.pdf
   - Contribution_Plan.pdf
   - LPDP_Full_Package_Muhammad_Sobri_Maulana.pdf

2. CV & Portfolio:
   - CV_Muhammad_Sobri_Maulana.pdf
   - Publication_List.pdf
   - Project_Portfolio.pdf

3. Sertifikat (Opsional):
   - ACLS_Certificate.pdf
   - ATLS_Certificate.pdf
   - [Sertifikat lainnya]

PANDUAN:
--------

1. Nama File:
   - Gunakan format: [Tipe_Dokumen]_[Nama].pdf
   - Contoh: CV_Muhammad_Sobri_Maulana.pdf
   - Hindari spasi, gunakan underscore (_)

2. Ukuran File:
   - Maksimal 5MB per file
   - Compress PDF jika terlalu besar
   - Tools: iLovePDF, Smallpdf, Adobe Acrobat

3. Format:
   - Format: PDF only
   - Pastikan teks dapat di-copy (bukan scan gambar murni)
   - Resolusi cukup untuk dibaca (tidak terlalu rendah)

4. Keamanan:
   - Jangan cantumkan NIK, STR, atau data sensitif
   - Alamat cukup kota/provinsi
   - Nomor telepon gunakan email profesional sebagai gantinya

5. Checklist Sebelum Upload:
   [ ] Nama dan gelar benar
   [ ] Tidak ada typo
   [ ] Data sensitif sudah dihapus
   [ ] Format PDF
   [ ] File dapat dibuka dan dibaca
   [ ] Ukuran < 5MB

CARA MENAMBAHKAN FILE:
---------------------

1. Siapkan dokumen PDF Anda
2. Copy file ke folder public/documents/
3. Pastikan nama file sesuai dengan yang di-link di website
4. Build dan deploy ulang website

Jika nama file berbeda dari yang ada di website, update link di:
- src/pages/Home.jsx
- src/pages/Profile.jsx
- src/pages/LPDPPackage.jsx
- src/pages/Publications.jsx
- src/pages/Projects.jsx
- src/pages/Documents.jsx
- src/pages/Contact.jsx
