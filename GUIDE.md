## ğŸ“¦ Cara Deploy

### 1. Install Dependencies
```bash
npm install
```

### 2. Test Local
```bash
npm run dev
```
Buka http://localhost:5173

### 3. Build Production
```bash
npm run build
```

### 4. Deploy ke Netlify

**Opsi A: Netlify CLI**
```bash
npm install -g netlify-cli
netlify login
netlify deploy --prod
```

**Opsi B: GitHub Integration**
1. Push ke GitHub
2. Connect repo di Netlify
3. Auto-deploy on push

**Opsi C: Drag & Drop**
1. Drag folder `dist` ke Netlify
