# Portofolio LPDP Onkologi Radiasi

Website ini menampilkan portofolio LPDP dr. Muhammad Sobri Maulana untuk aplikasi Program Pendidikan Dokter Spesialis Onkologi Radiasi. Kontennya mencakup profil, dokumen LPDP, publikasi, proyek, serta informasi kontak profesional dalam satu situs yang mudah ditinjau.

## Stack

- React
- Vite
- React Router
- Komponen JSX dan TSX yang dapat dikembangkan lebih lanjut

## Halaman Utama

1. **Beranda** — ringkasan profil, highlight aplikasi, dan akses cepat ke dokumen utama.
2. **Profil** — pendidikan, pengalaman, kompetensi, sertifikasi, dan prestasi.
3. **LPDP Package** — motivation letter, career plan, contribution plan, dan study plan.
4. **Publikasi & Riset** — karya ilmiah, konferensi, dan area riset.
5. **Proyek & Inovasi** — proyek kesehatan, inovasi AI, dan fokus pengembangan teknologi.
6. **Dokumen** — kumpulan dokumen penting yang dapat diunduh.
7. **Kontak** — informasi korespondensi profesional dan formulir pesan.

## Menjalankan Proyek

```bash
npm install
npm run dev
npm run build
npm run preview
```
